My projects

I work on many things. Some of the main ones you may be looking for are listed here:

  • Ace Library

    A library module for using the Ace editor in Foundry VTT

  • Journal Code Editor

    A module for Foundry VTT that allows you to modify the source code of your Journal Entries using the editor library of your choice.

  • Atomizer

    A build and extract tool for FoundryVTT database files

  • PF2e Sheet Toggle Banner

    A module for the Foundry VTT PF2E character sheet that adds a simple toggle to the red side banner

  • SCS

    An implementation of the Simultaneous Combat System for Foundry VTT

  • Default Context Menu

    Use the default browser context menu in Foundry VTT

  • Export Sheet to PDF

    A system agnostic tool to export your Foundry character sheet to a PDF!

  • Combat Tracker Groups

    Group combatants in the Combat Tracker and roll for group initiative.

  • Sidebar Macros

    Add Macros to the Foundry VTT sidebar!

  • World Smiths Toolkit

    A toolkit for use in World Smiths products.
